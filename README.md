# RememberMe
Website development based on PHP+MySQL+Bootstrap
## 设计理念
网站受到《寻梦环游记》的启发，秉持着“死亡不是生命的终点，遗忘才是”的理念，希望逝者能够通过长存于心而长存于世。
星球与玫瑰的灵感来源于《小王子》，将逝者化为天上的一颗星星，用玫瑰来寄托对逝者的缅怀。
